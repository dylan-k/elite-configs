

"Will it Steam Controller?" Config for Elite Damgerous
===============================================================================

<iframe width="560" height="315" src="https://www.youtube.com/embed/hYdbNg5lqPA" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

The configuration made for the ThatBengineer's YouTube show, "Will It Steam Controller?" 

Features Radial Menus for one-touch commands. Bindings include pitch and roll on joystick, yaw and up/down thrust on right trackpad, forward/back thrust and left/right thrust on right trackpad with mode shift.  Use right grip for gyro head look.


SETUP
--------------------------------------------------------------------------------

1. copy `wisc.vdf` to the correct Steam folder. Or you can use `steam://controllerconfig/359320/1380779410`
2. copy `wisc.binds` to the correct Elite Dangerous folder

see main README for setup details.
