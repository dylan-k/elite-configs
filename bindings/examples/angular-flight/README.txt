

"Angular Flight" Controller Config for Elite Damgerous
===============================================================================

Bindings for Steam Controller and also the HOTAS, Thrustmaster.T16000m. Designed to maximize immersion in VR. Uses Steam Controller's Gyro for precision aiming.

Mod of Mod of "Angular Flight Standard Pro." by Angular Velocity on Steam aka Reddit u/HeadBoy



Setup
--------------------------------------------------------------------------------

Copy `angular-flight_0.vdf` into `C:\Program Files (x86)\Steam\userdata\<youraccountid>\241100\remote\controller_config\<game_appid>` (or you can paste link into windows explorer and apply configuration in steam.
`steam://controllerconfig/359320/1928192545`)

Copy `angular-flight.3.0.binds` into "C:\Users\[yourname]\AppData\Local\Frontier Developments\Elite Dangerous\Options\Bindings"
