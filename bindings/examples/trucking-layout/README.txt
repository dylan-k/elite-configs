
Yeshe McGermots' trucking layout
===============================================================================

Yeshe McGermots' trucking layout is the "Official Elite Dangerous Bindings" plus action set toggle on left stick hold for additional important functions and touch menu on right pad. (Check configuration and make sure that your keyboard configuration matches!)



SETUP
--------------------------------------------------------------------------------

1. Copy `trucking-layout.vdf` to the correct Steam folder. (Or use `steam://controllerconfig/359320/1150798892`.)
2. Like the official bindings, this should work with any of the in-game presets?

See main README for details.
