

EDdiscovery Settings

Symlink the *edtheme file to:  
`%localappdata%\EDDiscovery\Theme`
and then you can select it as a theme in EDDiscovery.

Symlink the EDDuser.sqlite file to %localappdata%\eddiscovery

Symlink `\Exploration\`to 
`%localappdata%\EDDiscovery\Exploration`

Symlink `\Expeditions\`to 
`%localappdata%\EDDiscovery\Expeditions`
